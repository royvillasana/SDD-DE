# Spacing

The spacing scale drives both **padding** and **margin** in every direction. One step of the
scale is the base unit; utilities multiply it (`p-*` / `m-*`, and directional `pt/px/mt/mx/…`).

## Direction reference

| Prefix | Meaning | CSS |
| --- | --- | --- |
| `p-*` / `m-*` | All sides | `padding` / `margin` |
| `pt/mt` · `pb/mb` | Top · Bottom | `padding/margin-block` |
| `ps/ms` · `pe/me` | Start · End | `padding/margin-inline` |
| `px/mx` · `py/my` | Horizontal · Vertical | both axes |
| `mx-auto` | Center | `margin: 0 auto` |

## Usage guidelines

- Compose layouts from scale steps only — never arbitrary pixel values.
- Use consistent vertical rhythm: the same step between sibling blocks.
- `mx-auto` centers a fixed-width block within its container.
