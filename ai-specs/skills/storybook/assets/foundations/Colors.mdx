## Usage guidelines

- Use **Primary** for main calls-to-action, links, and interactive elements.
- Use **Success / Danger / Warning / Info** for feedback states and alerts.
- Use mid neutrals for body text, lighter neutrals for muted/helper text.
- Ensure a contrast ratio ≥ 4.5:1 for text on colored backgrounds (WCAG AA).
